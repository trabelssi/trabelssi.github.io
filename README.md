# 🚀 Mohamed Amine Trabelsi - Portfolio

A modern, futuristic portfolio website showcasing telecommunications engineering, embedded systems, and network infrastructure expertise.



## 🌟 Features

### ✨ Core Features
- **Responsive Design** - Works perfectly on desktop, tablet, and mobile
- **Smooth Animations** - Parallax effects, section reveals, and staggered animations
- **Custom Cursor** - Interactive cursor with particle trail effects
- **Loading Screen** - Professional animated loading with progress bar
- **Dark Theme** - Eye-friendly dark design with green accent color

### 🎨 Interactive Elements
- **Horizontal Project Carousel** - Drag-to-scroll project showcase
- **Scroll Progress Circle** - Real-time scroll tracking (bottom-left)
- **Section Navigation Dots** - Quick jump navigation (right side)
- **Magnetic Cards** - Service cards with magnetic hover effects
- **Animated Progress Bars** - Skills visualization with smooth animations
- **Project Filtering** - Filter projects by category (Network, AI/ML, Embedded, Full-Stack)

### 📊 Sections
1. **Hero** - Animated introduction with stats counter
2. **About** - Personal info with typing effect animation
3. **Services** - What I do with 3D tilt cards
4. **Experience** - Professional timeline
5. **Projects** - Horizontal scrolling showcase
6. **Tech Stack** - Categorized technologies
7. **Contact** - Professional contact card

## 🚀 Quick Start

### Local Development
1. Clone the repository:
   ```bash
   git clone https://github.com/trabelssi/TrabelsiMohamedAmine.github.io.git
   ```

2. Open `index.html` in your browser
   - **Recommended:** Use Live Server extension in VS Code
   - Or simply double-click `index.html`

3. Make your changes and test locally

### Deployment to GitHub Pages
1. Push your changes to GitHub:
   ```bash
   git add .
   git commit -m "Update portfolio"
   git push origin main
   ```

2. Enable GitHub Pages:
   - Go to repository **Settings** → **Pages**
   - Select **main** branch
   - Save and wait 2-3 minutes

3. Your site will be live at: `https://trabelssi.github.io/TrabelsiMohamedAmine.github.io/`

## 🎨 Customization

### Change Colors
Edit `assets/css/main.css` and update CSS variables:
```css
:root {
    --bg-dark: #0a0a0a;        /* Background color */
    --accent: #00ff00;          /* Primary accent (green) */
    --text-primary: #ffffff;    /* Main text color */
    --text-secondary: #888888;  /* Secondary text */
}
```

### Update Content
1. **Personal Info** - Edit the hero section in `index.html`
2. **Projects** - Update project cards in the horizontal scroll section
3. **Skills** - Modify tech stack categories
4. **Contact** - Update email, phone, and social links

### Add/Remove Sections
- Each `<section>` has an `id` attribute
- Update navigation links in the navbar
- Update scroll navigation dots in JavaScript

For detailed customization, see [docs/CUSTOMIZATION-GUIDE.md](docs/CUSTOMIZATION-GUIDE.md)
